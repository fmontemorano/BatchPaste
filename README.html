BatchPaste

  • What is it?

    • A macro for Google Spreadsheets written in Google Apps Script.

  • What is it for?

    • Efficiently copying and pasting cells.  

  • How does it work?

    • It uses batch operations to copy and paste.

  • How do I use it?

    • Use this code for one of your Google Spreadsheet (https://docs.google.com/spreadsheet) Scripts.
